# 🎲 Sorteador Digital

Ferramenta para sorteios aleatórios desenvolvida com Python e Flask.

## Funcionalidades

- Sorteio aleatório de nomes ou itens
- Interface simples e intuitiva
- Resultado instantâneo

## Tecnologias

- Python
- Flask
- HTML
- CSS

## Como executar

```bash
pip install -r requirements.txt
python app.py
```

## Estrutura

```text
sorteador-digital/
├── app.py
├── requirements.txt
├── templates/
│   └── index.html
└── static/
    └── style.css
```
